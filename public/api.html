Nanoblog API


Pobieranie danych przechowywanych w serwisie nanoblogo.herokuapp.com w formacie JSON

Serwis nanoblogo.herokuapp.com/api/ udostępnia publiczne Web API umożliwiające klientom HTTP wykonywanie zapytań na poniższych zbiorach danych publikowanych przez serwis nanoblogo.herokuapp.com:
 1. Użytkowników
 2. Postów użytkowników
 3. Komentarzy użytkowników

Komunikacja z serwisem polega na wysłaniu odpowiednio sparametryzowanego żądania HTTP GET na adres bazowy http://nanoblogo.herokuapp.com/api/.

Opis funkcji API dotyczących użytkowników

Wzory zapytań o użytkowników

  • Wszyscy użytkownicy
    • https://nanoblogo.herokuapp.com/api/users/
  • Użytkownik o konkretnym {objectId}
    • https://nanoblogo.herokuapp.com/api/users/{objectId}

Wzory zapytań o klasy bazy danych

Zarówno komentarze jak i posty użytkowników przechowywane są klasach bazy danych. Standardowym zapytaniem o wszystkie elementy klasy {class} jest:

https://nanoblogo.herokuapp.com/api/classes/{class}

Istnieje też możliwość zapytania o konkretny obiekt {objectId} klasy {class}. Zapytanie wygląda wtedy w następujący sposób:

https://nanoblogo.herokuapp.com/api/classes/{class}/{objectId}
